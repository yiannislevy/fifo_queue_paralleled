# FIFO Queue Parallelized with POSIX Threads
A FIFO pipe implemented in C, using p-threads to optimize the flow of items. Multiple producers and consumers add and remove items from the queue.

## Build
```bash
cd rtes
make compile
```
## Run
```bash
make run
```
